Games

I have worked on several games, some of which were part of my university studies. Each project has taught me about development pipelines, workflows and common practices in the games industry and has pushed me to learn new skills to solve implementation challenges.

Rhythm Warriors

A rhythm game where characters fight in time with the music. Created entirely on my own for my Year 12 Industrial Technology: Multimedia major work.

Starship Scramble

An interstellar salvaging mission where players pilot starships through dangerous levels using strange controls. Created with two other students for the subject Game Design Methodologies at UTS.

Goodnight, MiMi!

A floaty puzzle launching game where one wrong move makes you lose a lot of progress. I'm pretty sure it's impossible to beat. Created as part of a team for the Ludwig Jam 2021.

Samurai Pac

A Pac-Man clone with added bullet-deflecting mechanics. Created on my own for the subject Introduction to Computer Game Development at UTS.

Jetski Joyride

A side-scrolling dodging game where you clean the oceans to save the turtles. Created as part of a team for the Playmakers Summertime Game Jam 2022.

Programs

These are projects involving computer software that weren't made for entertainment purposes. They were either produced for a university assignment, or as a hobby.

ArcticSeeder

A Node.js Discord bot designed to create matches and seed players for the team-based game Splatoon 3. Was used to draft players for the Frostbite League.

Dragon Isles

A Three.js program that generates floating islands and its inhabiting dragons. Created with a group of people for a Computer Graphics assignment at UTS.

Double Helix

A procedurally generated double-helix that can be customised using a bunch of parameters. Created in Three.js for a Computer Graphics assignment at UTS.

CampMaker

A sandbox-style environment with procedurally generated terrain that can be customised. Created in Three.js for a Computer Graphics assignment at UTS.
